# swagger-practice

- 스프링부트 프로젝트에 swagger 연동 practice
- http://localhost:8080/swagger-ui/index.html


### 개발환경
- Spring boot 2.7
- java 8
- springdoc-openapi-ui version: '1.7.0'

### build.gradle
- dependencies

`
depencies{
    implementation group: 'org.springdoc', name: 'springdoc-openapi-ui', version: '1.7.0'
}
`
